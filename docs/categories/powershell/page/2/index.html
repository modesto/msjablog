[PowerShell] Crear el fichero de manifiesto para muchos módulos a la vez

Hoy he tenido que crear de golpe el manifest a varios módulos. Teníamos más de 10 módulos (.psm1), cada uno en su propia carpeta y necesitaba añadirles el manifest para poder definir la empresa de cada módulo. Gracias al pipeline de PowerShell, no he tenido que esforzarme mucho para conseguirlo: ls *.psm1 -Recurse | ForEach-Object { $manifestFileName = $_.BaseName + ".psd1" $manifestPath = Join-Path $_.Directory $manifestFileName $companyConstant = "AnyCompany" New-ModuleManifest -Path $manifestPath -RootModule $_. »
